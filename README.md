#  My Green Nest:

   Green Nest Project is your go-to platform for bringing nature indoors. We help users discover and buy indoor plants tailored to their personal preferences, space, and lifestyle. By combining convenience with greenery, we make it easy for everyone to create a healthier, more vibrant living space. Our mission is to inspire a love for plants while promoting a sustainable and eco-friendly lifestyle.

##  Technologies Used

###  Core
- **React 19** – Component-based UI library  
- **React DOM** – For rendering React components to the browser  
- **Vite 7** – Lightning-fast build tool and development server  

---

###  UI & Styling
- **Tailwind CSS 4** – Utility-first CSS framework for rapid UI development  
- **DaisyUI 5** – Tailwind-based UI component library  
- **Framer Motion 12** – For beautiful animations and transitions  
- **React Icons 5** – Icon library for React  

---

###  Authentication & Database
- **Firebase 12** – Backend-as-a-service for authentication, hosting, and database  

---

###  Routing & Navigation
- **React Router 7** – Client-side routing for seamless navigation  

---

###  Notifications
- **React Toastify 11** – Elegant toast notifications for user feedback  
- **SweetAlert** – For beautiful alert and confirmation dialogs 

---

###  UI Enhancements
- **Swiper 12** – Modern slider/carousel library  

---

###  Developer Tools
- **ESLint 9** – Linting and code quality tool  
- **@vitejs/plugin-react** – React plugin for Vite  
- **TypeScript React Types** – Type definitions for React and DOM  
- **Globals** – Provides global variables for linting  
- **React Hooks & Refresh ESLint Plugins** – For better React linting experience
